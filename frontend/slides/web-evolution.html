Evolución e historia de Internet


Wikipedia: Historia de Internet
Para mayor información podéis utilizar el siguiente link que explica la evolución y las diferentes tecnologías relacionadas con el universo web: Evolución de la web o la línea temporal de los navegadores web en Wikipedia



Trident de Microsoft se usó en todos los IE hasta la versión 9. Las versiones 10 y superiores se basan en Lynx. No soportaba estándares web. La representación del mismo código en el resto de motores era inconsistente.


Gecko, creado para Netscape Navigator 1.0 en diciembre de 1994. Mejorado por Mozilla Foundation y ya con el nombre de Gecko, pasó a formar parte de Firebird y posteriormente de Firefox y el cliente de correo Thunderbird. Multiplataforma y web standards compliant.


Webkit, basado en Konkeror de KDE Linux. Modificado por Apple e introducido en Safari en 2003. A partir de entonces deja de ser Software Libre.


Edge, el sucesor de Trident. Desarrollado con el code name Spartan. Mejor rendimiento, mayor soporte de estándares web y disponible con Windows 10.


Blink, desarrollado por Google en 2013 como parte del proyecto open source Chromium. Pasa a formar parte de Chrome v28 y Opera v15. Una de sus características es que mejora el multi-proceso y pretende acabar con los vendor prefixes.

Microsoft confirmó antes de terminar el año que abandonarán el navegador Edge para basarse en Chromium. Al igual que Google Chrome, Opera, Vivaldi y Brave.
