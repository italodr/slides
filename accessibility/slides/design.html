Diseño accesible

El texto de primer plano debe tener suficiente contraste con los colores de fondo. Esto incluye texto en imágenes, gradientes de fondo, botones y otros elementos.

Si bien el color puede ser útil para transmitir información, el color no debe ser la única forma de transmitir la información.

Proporciona estilos distintos para elementos interactivos, como enlaces y botones, para que sean fáciles de identificar. Asegúrate que los estilos y nombres de los elementos interactivos sean coherentes en todo el sitio web.


Asegúrate que la navegación a través de las páginas dentro de un sitio web tenga nombres, estilos y posicionamientos consistentes. Proporciona más de un método de navegación del sitio web, como un buscador o un mapa del sitio. Ayuda a los usuarios a comprender dónde se encuentran en un sitio web brindando pistas de orientación, como breadcrumbs y encabezados claros.

  • Títulos y descripciones para tablas o gráficos complejos.
