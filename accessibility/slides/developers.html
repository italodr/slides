Añade texto alternativo para todas las imágenes informativas y funcionales. Usa el texto alternativo vacío alt="" para imágenes decorativas.

Indica el idioma principal de cada página utilizando el atributo lang en la etiqueta html, por ejemplo, <html lang="en">. Use el atributo lang en elementos específicos cuando el idioma del elemento difiera del resto de la página.

Usa el lenguaje de marcado o HTML adecuado para encabezados, listas, tablas, etc. HTML5 proporciona elementos adicionales, como <nav> y <aside>, para estructurar mejor su contenido. Los roles WAI-ARIA pueden proporcionar un significado adicional, por ejemplo, usar role="search" para identificar la funcionalidad de búsqueda.
